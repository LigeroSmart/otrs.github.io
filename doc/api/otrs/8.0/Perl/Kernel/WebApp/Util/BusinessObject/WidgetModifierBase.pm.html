NAME

Kernel::WebApp::Util::BusinessObject::WidgetModifierBase

DESCRIPTION

Base class for widget modifier modules. Widget modifiers are used to possibly modify widget configurations right before they will be provided to the frontend and/or right before they are stored to the database.

PUBLIC INTERFACE

ConfigGet()

This method will be called right before the related widget configuration is provided to the frontend.

ConfigSet()

This method will be called right before the related widget configuration is stored in the database.
