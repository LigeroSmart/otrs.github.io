担当者ウェブインターフェイス

担当者は、担当者ウェブインターフェイスで顧客のリクエストに答えることができます。顧客のためあるいは他の担当者のために新規チケットを作成し、顧客との通話に関することをチケットに書き、FAQを記入し、顧客データを編集します。

URL http://www.example.com 経由であなたのOTRSホストへ到達すると仮定すれば、ウェブブラウザ（下図参照）でアドレス http://www.example.com/otrs/index.pl を使用することでOTRSログイン画面へ到達します。
