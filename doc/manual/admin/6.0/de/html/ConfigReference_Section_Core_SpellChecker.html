Core::SpellChecker

SpellChecker

Aktiviert die Unterstützung für die Rechtschreibprüfung.

Diese Einstellung kann nicht deaktiviert werden.

Standardwert: 

--- 0

SpellCheckerBin

Install ispell or aspell on the system, if you want to use a spell checker. Please specify the path to the aspell or ispell binary on your operating system.

Diese Einstellung kann nicht deaktiviert werden.

Standardwert: 

--- /usr/bin/ispell

SpellCheckerDictDefault

Definiert das Standard Rechtschreib-Wörterbuch

Diese Einstellung kann nicht deaktiviert werden.

Standardwert: 

--- english

SpellCheckerIgnore

Definiert eine Standardliste von Wörtern, die von der Rechtschreibprüfung ignoriert werden.

Diese Einstellung kann nicht deaktiviert werden.

Standardwert: 

---
- www
- webmail
- https
- http
- html
- rfc
