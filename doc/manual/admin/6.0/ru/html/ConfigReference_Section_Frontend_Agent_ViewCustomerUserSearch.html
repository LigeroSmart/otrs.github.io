Frontend → Agent → ViewCustomerUserSearch

CustomerUser::Frontend::AgentCustomerUserAddressBook###DefaultFields

Defines the default filter fields in the customer user address book search (CustomerUser or CustomerCompany). For the CustomerCompany fields a prefix 'CustomerCompany_' must be added.

Этот параметр не может быть отключен.

Значение по умолчанию: 

---
- UserLogin
- UserFirstname
- UserLastname
- UserEmail
- UserCustomerID

CustomerUser::Frontend::AgentCustomerUserAddressBook###PageShown

Defines the number of entries per page for the AgentCustomerUserAddressBook result screen.

Этот параметр не может быть отключен.

Значение по умолчанию: 

--- '50'

CustomerUser::Frontend::AgentCustomerUserAddressBook###SearchLimit

Defines the search limit for the AgentCustomerUserAddressBook screen.

Этот параметр не может быть отключен.

Значение по умолчанию: 

--- '10000'

CustomerUser::Frontend::AgentCustomerUserAddressBook###ShowColumns

Defines the shown columns and the position in the AgentCustomerUserAddressBook result screen.

Этот параметр не может быть отключен.

Значение по умолчанию: 

---
- UserFirstname
- UserLastname
- UserLogin
- UserEmail
- UserCustomerID
