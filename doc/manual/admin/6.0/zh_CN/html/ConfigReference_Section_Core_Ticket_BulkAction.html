Core → Ticket → BulkAction

Ticket::Frontend::BulkFeature

启用工单批量操作，以在服务人员前端一次性操作多个工单。

不能使这个设置失效。

默认值: 

--- '1'

Ticket::Frontend::BulkFeatureGroup

仅对列表中的组启用批量操作功能。

这个设置默认没有激活。

默认值: 

---
- admin
- users
