Core::SpellChecker

SpellChecker

启用拼写检查。

不能使这个设置失效。

默认值: 

--- 0

SpellCheckerBin

如果您想使用拼写检查器，请在系统中安装ispell 或 aspell。请指定ispell 或 aspell在操作系统中的程序路径。

不能使这个设置失效。

默认值: 

--- /usr/bin/ispell

SpellCheckerDictDefault

定义默认的拼写检查词典。

不能使这个设置失效。

默认值: 

--- english

SpellCheckerIgnore

定义一个默认的词语列表，拼写检查器会忽略这个列表中的词语。

不能使这个设置失效。

默认值: 

---
- www
- webmail
- https
- http
- html
- rfc
