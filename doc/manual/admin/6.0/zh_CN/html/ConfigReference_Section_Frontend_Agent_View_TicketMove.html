Frontend → Agent → View → TicketMove

Ticket::Frontend::AgentTicketMove###Body

设置服务人员界面转移工单窗口添加备注的的默认正文文本。

This setting can be overridden in the user preferences, but is not active by default.

默认值: 

--- ''

Ticket::Frontend::AgentTicketMove###DynamicField

Dynamic fields shown in the ticket move screen of the agent interface.

默认值: 

--- {}

Ticket::Frontend::AgentTicketMove###NextScreen

确定工单转移队列后的下一个窗口。LastScreenOverview将返回到最后的概览窗口（例如：搜索结果、队列视图、仪表板），TicketZoom将返回到工单详情视图。

不能使这个设置失效。

默认值: 

--- TicketZoom

Ticket::Frontend::AgentTicketMove###Note

在服务人员界面允许在工单自定义字段界面添加备注，可以被Ticket::Frontend::NeedAccountedTime设置覆盖。

This setting can be overridden in the user preferences, but is not active by default.

默认值: 

--- 0

Ticket::Frontend::AgentTicketMove###NoteMandatory

设置是否必须由服务人员填写备注，能够被参数Ticket::Frontend::NeedAccountedTime覆盖。

This setting can be overridden in the user preferences, but is not active by default.

默认值: 

--- 0

Ticket::Frontend::AgentTicketMove###Priority

在服务人员界面转移工单窗口是否显示工单优先级的选项。

不能使这个设置失效。

默认值: 

--- 0

Ticket::Frontend::AgentTicketMove###RequiredLock

在服务人员界面处理转移工单后自动锁定并设置当前服务人员为工单所有者。

默认值: 

--- '1'

Ticket::Frontend::AgentTicketMove###RichTextHeight

定义本窗口富文本编辑器组件的高度。输入数值（像素值）或百分比值（相对值）。

This setting can be overridden in the user preferences, but is not active by default.

默认值: 

--- '100'

Ticket::Frontend::AgentTicketMove###RichTextWidth

定义本窗口中富文本编辑器组件的宽度。输入数值（像素值）或百分比值（相对值）。

This setting can be overridden in the user preferences, but is not active by default.

默认值: 

--- '620'

Ticket::Frontend::AgentTicketMove###State

在服务人员界面允许在转移工单界面设置新的工单状态。

This setting can be overridden in the user preferences, but is not active by default.

默认值: 

--- '1'

Ticket::Frontend::AgentTicketMove###StateMandatory

Sets if state must be selected by the agent.

默认值: 

--- 0

Ticket::Frontend::AgentTicketMove###StateType

定义服务人员界面转移工单窗口转移工单到另一队列后的下一个工单状态。

不能使这个设置失效。

默认值: 

---
- open
- closed

Ticket::Frontend::AgentTicketMove###Subject

设置服务人员界面转移工单窗口添加备注的默认主题。

This setting can be overridden in the user preferences, but is not active by default.

默认值: 

--- ''

Ticket::Frontend::MoveType

确定在服务人员界面转移工单到可能的队列列表是否在下拉列表中或新窗口显示。如果设置为“新窗口”，您可以为这个工单添加一个移动备注。

不能使这个设置失效。

默认值: 

--- form
