バグ・トラッキング

バグを送信するためにhttp://bugs.otrs.org/を訪ねて下さい(下記の図を参照)。バグと構成問題間の違いについてノートをとってください。構成問題は、システムを設定する時遭遇する問題、あるいはOTRSの使用に関する一般的な質問です。バグ・レポートは、OTRSのソース・コード自体の問題あるいはOTRSの改善をファイルするためのみに使用されるべきです。全てのバグ・レポートおよび改善要求を、バグ・トラッカー非常に歓迎します。

図：バグ・トラッカー

構成問題については、OTRS.comから利用可能な商用サポート、あるいは公式メーリング・リストのどちらかを利用すべきです。

みなさんのバグ・レポートによって製品が改善されます。みなさんのインプットに感謝します！
