Пользовательский веб-интерфейс

В OTRS клиенты имеют отдельный веб-интерфейс, с помощью этого веб-интерфейса они могут создавать новые учетные записи, просматривать собственные заявки, создавать и редактировать заявки, изменять настройки своей учетной записи и т.д.

Используя выше приведенный пример, страница входа для клиентов будет доступна по ссылке http://www.example.com/otrs/customer.pl (см. Рисунок ниже).
