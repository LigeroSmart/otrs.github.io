Общедоступный веб-интерфейс

В дополнение к веб-интерфейсу для агентов и клиентов, OTRS имеет открытый веб-интерфейс, предоставляющий доступ к FAQ-модулю, который устанавливается отдельно. Он предоставляет открытый доступ к FAQ-системе и позволяет пользователям без авторизации искать информацию в базе знаний.

В нашем примере общий веб-интерфейс будет доступен по ссылке: http://www.example.com/otrs/faq.pl , http://www.example.com/otrs/public.pl
