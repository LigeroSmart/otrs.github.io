OTRSフロント・エンドのローカライゼーション

OTRSはそのウェブ・インターフェースのために多重言語サポートを提供します。

OTRSフレームワーク用ローカライゼーションのための手続き、新しい言語変換を作成するために続くステップ、と同様に、翻訳カスタマイズのための手続き、開発者マニュアルhttp://otrs.github.io/doc の "Language Translations（言語変換）"章で見つけることができます。
