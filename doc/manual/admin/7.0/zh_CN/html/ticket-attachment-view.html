The Article Attachment Overview

描述
配置
用法

描述

The article attachment overview enables you to view a list of all attachments in a ticket. Attachments will be listed by attributes such as file name, file type, file size, date of addition and which article they belong to. From the overview screen, you can download certain attachments or create an archive file which contains multiple or all attachments of the ticket you're viewing.

配置

No further configuration needed.

用法

图 6.44. Switching to the article attachment overview


Once a ticket has at least one attachment, a new menu entry, Attachments, will appear under the 'Miscellaneous' menu cluster. Clicking it will open a popup which contains a list of all attachments.
