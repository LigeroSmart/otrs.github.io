Frontend → External → PrivacyNotice

ExternalFrontend::PrivacyNotice###de

Defines privacy notice configuration for the external interface. The notice will be shown to all users until they accept the privacy policy.

默认值: 

---
ButtonText: OK, got it!
Link: https://www.example.com/your-cookie-consent
LinkTarget: _blank
LinkText: data protection
Text: This site uses cookies. By clicking ok you accept our %s agreement.
