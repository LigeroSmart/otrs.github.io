Core → Chat

ChatEngine::Active

Activates chat support.

不能使这个设置失效。

默认值: 

--- 0

ChatEngine::ChatDecayTime

Number of days after chat will be deleted.

默认值: 

--- '365'

ChatEngine::ChatDirection::AgentToAgent

Makes it possible to start a chat with an agent from the agent interface.

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatDirection::AgentToCustomer

Makes it possible to start a chat with a customer from the agent interface.

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatDirection::CustomerToAgent

Makes it possible to start a chat with an agent from the customer interface.

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatDirection::PublicToAgent

Makes it possible to start a chat with an agent from the public interface.

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatOrder

Defines the order of chat windows.

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatTTL

Number of hours after closed chat will be deleted.

默认值: 

--- '24'

ChatEngine::CustomerFrontend::AgentPreferences

Defines a list of agent preferences group keys that will be visible in Customer interface (works only if DefaultAgentName is disabled). Example: If you want to display PreferencesGroups###Language, add Language.

不能使这个设置失效。

默认值: 

--- []

ChatEngine::CustomerFrontend::AgentPreferencesText

Defines a text for Agent Preferences in the Customer interface (works only if DefaultAgentName is disabled).

不能使这个设置失效。

默认值: 

--- Agent Information

ChatEngine::CustomerFrontend::AvailabilityCheck

Allow customer to select only channels that have available agent(s).

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::CustomerInterface::AllowChatChannels

Defines if customer users are able to select Chat Channel. If not, chat will be created in default Chat Channel.

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::CustomerThreshold

Defines the period of time (in minutes) before No Answer message is displayed to the customer.

不能使这个设置失效。

默认值: 

--- '5'

ChatEngine::DefaultAgentName

Default agent name for customer and public interface. If enabled, the real name of the agent will not be visible to customers/public users while using the chat.

这个设置默认没有激活。

默认值: 

--- Support Agent

ChatEngine::DefaultAgentNameNumbers

Defines if numbers should be appended to DefaultAgentName. If enabled, together with DefaultAgentName will be numbers (e.g. 1,2,3,...).

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::DefaultChatChannel

Name of default chat channel. If this channel does not exist, it will be created automatically. Please do not create a chat channel with the same name as default chat channel. The default channel won't be displayed, if chat channels are enabled in Customer interface and Public interface. All agent to agent chats will be in the default channel.

不能使这个设置失效。

默认值: 

--- Default channel

ChatEngine::PermissionGroup::ChatReceivingAgents

Agent group that can accept chat requests and chat.

不能使这个设置失效。

默认值: 

--- users

ChatEngine::PermissionGroup::ChatStartingAgents

Agent group that can create chat requests.

不能使这个设置失效。

默认值: 

--- users

ChatEngine::PermissionGroup::VideoChatAgents

Agent group that can use video calling feature in chats.

不能使这个设置失效。

默认值: 

--- users

ChatEngine::PublicInterface::AllowChatChannels

Defines if public users are able to select Chat Channel. If not, chat will be created in default Chat Channel.

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::PublicInterface::AvailabilityCheck

Allow public users to select only channels that have available agent(s).

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::Texts::CustomerFrontend::ChatClosed

Default text for the customer interface.

不能使这个设置失效。

默认值: 

--- This chat has ended. Please leave it or it will be closed automatically. You can
  download this chat.

ChatEngine::Texts::CustomerFrontend::ChatCompleted

Default text for the customer interface.

不能使这个设置失效。

默认值: 

--- This chat is completed. You can now close the chat window.

ChatEngine::Texts::CustomerFrontend::NoAgentsAvailable::AddToExisiting

No agents available message.

不能使这个设置失效。

默认值: 

--- 'There are no chat agents available at present. To add an article to the existing
  ticket, please click here:'

ChatEngine::Texts::CustomerFrontend::NoAgentsAvailable::NewTicket

No agents available message.

不能使这个设置失效。

默认值: 

--- 'There are no chat agents available at present. To create a new ticket, please
  click here:'

ChatEngine::Texts::CustomerFrontend::NotYetAccepted

Default text for the customer interface.

不能使这个设置失效。

默认值: 

--- Please bear with us until one of our agents is able to handle your chat request.
  Thank you for your patience.

ChatEngine::Texts::PublicFrontend::ChatCompleted

Default text for the public interface.

不能使这个设置失效。

默认值: 

--- This chat is completed. You can start a new chat using the button on top of this
  page.

ChatEngine::Texts::PublicFrontend::NoAgentsAvailable

Default text for the public interface.

不能使这个设置失效。

默认值: 

--- There are no chat agents available at present. Please try again later.

ChatEngine::Texts::PublicFrontend::NotYetAccepted

Default text for the public interface.

不能使这个设置失效。

默认值: 

--- Please bear with us until one of our agents is able to handle your chat request.
  Thank you for your patience.

ChatEngine::Texts::PublicInterface::ChatClosed

Default text for the public interface.

不能使这个设置失效。

默认值: 

--- This chat has ended. Please leave it or it will be closed automatically.

Ticket::Agent::DisplayNotificationIfUnavailable

If enabled, a notification will be displayed on each page if the current agent is not available for chat.

默认值: 

--- '1'

Ticket::Agent::StartChatFromTicket###AllowChatOnlyIfCustomerAvailable

If enabled, starting a chat from agent ticket zoom view will only be possible, if the ticket customer is on-line.

默认值: 

--- '1'

Ticket::Agent::StartChatFromTicket###Allowed

Sets if a chat request can be sent out from agent ticket zoom view.

默认值: 

--- '1'

Ticket::Agent::StartChatFromTicket###PermissionLevel

Permission level to start a chat with customer from agent ticket zoom view.

默认值: 

--- ro

Ticket::Agent::StartChatFromTicket###RequiredLock

Defines if locking a ticket is required to start a chat with the customer from the ticket zoom view.

默认值: 

--- 0

Ticket::Agent::StartChatWOTicket

If enabled, agents can start a chat with a customer without a ticket.

默认值: 

--- '1'

Ticket::Agent::UnavailableForExternalChatsOnLogin

If enabled it will check agents availability on login. If user is available for external chats, it will reduce availability to internal chats only.

默认值: 

--- '1'

Ticket::Customer::StartChatFromTicket###AllowChatOnlyIfAgentsAvailable

If enabled, starting a chat from customer ticket zoom view will only be possible, if there are available agents in the linked chat channel.

默认值: 

--- '1'

Ticket::Customer::StartChatFromTicket###Allowed

Sets if a chat request can be sent out from customer ticket zoom view.

默认值: 

--- '1'

Ticket::Customer::StartChatFromTicket###Permissions

If enabled, only the ticket customer can start a chat from a ticket zoom view.

默认值: 

--- 0

Ticket::Customer::StartChatWOTicket

If enabled, customers can start a chat with an agent without a ticket.

默认值: 

--- '1'
