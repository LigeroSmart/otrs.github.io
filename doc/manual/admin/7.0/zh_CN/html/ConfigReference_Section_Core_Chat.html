Core → Chat

ChatEngine::Active

激活聊天支持

不能使这个设置失效。

默认值: 

--- 0

ChatEngine::ChatDecayTime

聊天删除前保留的天数。

默认值: 

--- '365'

ChatEngine::ChatDirection::AgentToAgent

从服务人员界面可以发起与一个服务人员的聊天。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatDirection::AgentToCustomer

从服务人员界面可以发起与一个客户的聊天。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatDirection::CustomerToAgent

从客户界面可以发起与一个服务人员的聊天。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatDirection::PublicToAgent

从公共用户界面可以发起与一个服务人员的聊天。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatOrder

定义聊天窗口的顺序。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::ChatTTL

已关闭的聊天删除前保留的小时数。

默认值: 

--- '24'

ChatEngine::CustomerFrontend::AgentPreferences

定义一个客户界面可见的服务人员首选项组键的列表（仅当禁用DefaultAgentName才有效）。例如：你要显示PreferencesGroups###Language，添加语言。

不能使这个设置失效。

默认值: 

--- []

ChatEngine::CustomerFrontend::AgentPreferencesText

定义客户界面服务人员首选项的一段文本（仅当禁用DefaultAgentName才有效）。

不能使这个设置失效。

默认值: 

--- Agent Information

ChatEngine::CustomerFrontend::AvailabilityCheck

允许客户只选择有可用服务人员的频道。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::CustomerInterface::AllowChatChannels

定义客户用户能否选择聊天频道。如果不能，就在默认聊天频道创建聊天。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::CustomerThreshold

定义显示“无应答”消息给客户的时间段（分钟）。

不能使这个设置失效。

默认值: 

--- '5'

ChatEngine::DefaultAgentName

客户和公共用户界面看到的默认服务人员姓名。如果启用了，聊天时客户/公共用户无法知晓服务人员的真实姓名。

这个设置默认没有激活。

默认值: 

--- Support Agent

ChatEngine::DefaultAgentNameNumbers

定义是否在默认服务人员姓名后追加数字，如果启用，将在默认服务人员姓名后加上数字（如1、2、3……）。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::DefaultChatChannel

默认的聊天频道名称。如果该频道不存在，将自动创建它。请不要创建一个与默认聊天频道相同名称的聊天频道。如果客户界面和公共用户界面启用了聊天频道，默认频道不会显示出来。所有的服务人员之间的聊天都在默认频道中。

不能使这个设置失效。

默认值: 

--- Default channel

ChatEngine::PermissionGroup::ChatReceivingAgents

能够接受聊天请求和聊天的服务人员组。

不能使这个设置失效。

默认值: 

--- users

ChatEngine::PermissionGroup::ChatStartingAgents

能够创建聊天请求的服务人员组。

不能使这个设置失效。

默认值: 

--- users

ChatEngine::PermissionGroup::VideoChatAgents

在聊天中能够使用视频通话功能的服务人员组。

不能使这个设置失效。

默认值: 

--- users

ChatEngine::PublicInterface::AllowChatChannels

定义公共用户能否选择聊天频道。如果不能，就在默认聊天频道创建聊天。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::PublicInterface::AvailabilityCheck

允许公共用户只选择有可用服务人员的频道。

不能使这个设置失效。

默认值: 

--- '1'

ChatEngine::Texts::CustomerFrontend::ChatClosed

客户界面的默认文本。

不能使这个设置失效。

默认值: 

--- This chat has ended. Please leave it or it will be closed automatically. You can
  download this chat.

ChatEngine::Texts::CustomerFrontend::ChatCompleted

客户界面的默认文本。

不能使这个设置失效。

默认值: 

--- This chat is completed. You can now close the chat window.

ChatEngine::Texts::CustomerFrontend::NoAgentsAvailable::AddToExisiting

没有服务人员可用的消息。

不能使这个设置失效。

默认值: 

--- 'There are no chat agents available at present. To add an article to the existing
  ticket, please click here:'

ChatEngine::Texts::CustomerFrontend::NoAgentsAvailable::NewTicket

没有服务人员可用的消息。

不能使这个设置失效。

默认值: 

--- 'There are no chat agents available at present. To create a new ticket, please
  click here:'

ChatEngine::Texts::CustomerFrontend::NotYetAccepted

客户界面的默认文本。

不能使这个设置失效。

默认值: 

--- Please bear with us until one of our agents is able to handle your chat request.
  Thank you for your patience.

ChatEngine::Texts::PublicFrontend::ChatCompleted

公共用户界面的默认文本。

不能使这个设置失效。

默认值: 

--- This chat is completed. You can start a new chat using the button on top of this
  page.

ChatEngine::Texts::PublicFrontend::NoAgentsAvailable

公共用户界面的默认文本。

不能使这个设置失效。

默认值: 

--- There are no chat agents available at present. Please try again later.

ChatEngine::Texts::PublicFrontend::NotYetAccepted

公共用户界面的默认文本。

不能使这个设置失效。

默认值: 

--- Please bear with us until one of our agents is able to handle your chat request.
  Thank you for your patience.

ChatEngine::Texts::PublicInterface::ChatClosed

公共用户界面的默认文本。

不能使这个设置失效。

默认值: 

--- This chat has ended. Please leave it or it will be closed automatically.

Ticket::Agent::DisplayNotificationIfUnavailable

If enabled, a notification will be displayed on each page if the current agent is not available for chat.

默认值: 

--- '1'

Ticket::Agent::StartChatFromTicket###AllowChatOnlyIfCustomerAvailable

If enabled, starting a chat from agent ticket zoom view will only be possible, if the ticket customer is on-line.

默认值: 

--- '1'

Ticket::Agent::StartChatFromTicket###Allowed

设置服务人员工单详情视图能否发出聊天请求。

默认值: 

--- '1'

Ticket::Agent::StartChatFromTicket###PermissionLevel

从服务人员工单详情视图发起一个与客户的聊天所需的权限级别。

默认值: 

--- ro

Ticket::Agent::StartChatFromTicket###RequiredLock

定义从工单详情视图中开始与客户聊天时是否必须锁定工单。

默认值: 

--- 0

Ticket::Agent::StartChatWOTicket

If enabled, agents can start a chat with a customer without a ticket.

默认值: 

--- '1'

Ticket::Agent::UnavailableForExternalChatsOnLogin

如果启用本选项，将在登录时检查服务人员的可用性。如果用户可用外部聊天，就仅减少内部聊天的可用性。

默认值: 

--- '1'

Ticket::Customer::StartChatFromTicket###AllowChatOnlyIfAgentsAvailable

If enabled, starting a chat from customer ticket zoom view will only be possible, if there are available agents in the linked chat channel.

默认值: 

--- '1'

Ticket::Customer::StartChatFromTicket###Allowed

设置客户工单详情视图能否发出聊天请求。

默认值: 

--- '1'

Ticket::Customer::StartChatFromTicket###Permissions

If enabled, only the ticket customer can start a chat from a ticket zoom view.

默认值: 

--- 0

Ticket::Customer::StartChatWOTicket

If enabled, customers can start a chat with an agent without a ticket.

默认值: 

--- '1'
