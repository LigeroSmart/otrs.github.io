Frontend → External → View → TicketCreate

ExternalFrontend::TicketCreate###DynamicField

Dynamic fields options shown in the ticket message screen of the customer interface. NOTE. If you want to display these fields also in the ticket detail view of the external interface, you have to enable them in ExternalFrontend::TicketDetailView###DynamicField.

デフォルト値: 

--- {}

ExternalFrontend::TicketCreate###HistoryComment

顧客インタフェースの新規履歴エントリーのためのコメントです。

This setting can not be deactivated.

デフォルト値: 

--- ''

ExternalFrontend::TicketCreate###HistoryType

顧客インタフェースのデフォルトの履歴タイプを定義します。

This setting can not be deactivated.

デフォルト値: 

--- WebRequestCustomer

ExternalFrontend::TicketCreate###NextScreenAfterNewTicket

顧客インタフェースで、新規顧客チケットの後の次の画面を決定します。

This setting can not be deactivated.

デフォルト値: 

--- TicketOverview

ExternalFrontend::TicketCreate###Priority

顧客が顧客インタフェースでチケット優先度を設定することを、許可します。

This setting can not be deactivated.

デフォルト値: 

--- '1'

ExternalFrontend::TicketCreate###PriorityDefault

顧客インタフェースで、新規顧客チケットのデフォルトの優先度を定義します。

This setting can not be deactivated.

デフォルト値: 

--- 3 normal

ExternalFrontend::TicketCreate###Queue

Allows customers to set the ticket queue in the customer interface. If this is not enabled, QueueDefault should be configured.

This setting can not be deactivated.

デフォルト値: 

--- '1'

ExternalFrontend::TicketCreate###QueueDefault

顧客インタフェースで、新規顧客チケットのためのデフォルトのキューを定義します。

This setting is not active by default.

デフォルト値: 

--- Postmaster

ExternalFrontend::TicketCreate###SLA

顧客が顧客インタフェースでチケットSLAを設定することを、許可します。

This setting can not be deactivated.

デフォルト値: 

--- '1'

ExternalFrontend::TicketCreate###SLAMandatory

SLAが顧客によって必ず選択される必要があるかどうかを設定します。

デフォルト値: 

--- 0

ExternalFrontend::TicketCreate###SenderType

顧客インタフェースからの新規チケットのための送信者タイプです。

This setting can not be deactivated.

デフォルト値: 

--- customer

ExternalFrontend::TicketCreate###Service

顧客が顧客インタフェースでチケット・サービスを設定することを、許可します。

This setting can not be deactivated.

デフォルト値: 

--- '1'

ExternalFrontend::TicketCreate###ServiceMandatory

サービスが顧客によって必ず選択される必要があるかどうかを設定します。

デフォルト値: 

--- 0

ExternalFrontend::TicketCreate###StateDefault

顧客インタフェースで、新規顧客チケットのデフォルトの状態を定義します。

This setting can not be deactivated.

デフォルト値: 

--- new

ExternalFrontend::TicketCreate###TicketType

Allows customers to set the ticket type in the customer interface. If this is not enabled, TicketTypeDefault should be configured.

This setting can not be deactivated.

デフォルト値: 

--- '1'

ExternalFrontend::TicketCreate###TicketTypeDefault

Defines the default ticket type for new customer tickets in the customer interface.

This setting is not active by default.

デフォルト値: 

--- Unclassified
