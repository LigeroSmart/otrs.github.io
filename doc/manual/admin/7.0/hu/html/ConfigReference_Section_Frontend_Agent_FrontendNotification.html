Frontend::NotifyModule###9000-Generic

Defines the module that shows a generic notification in the agent interface. Either "Text" - if configured - or the contents of "File" will be displayed. Use "Priority" key to style the notification. If "Link" key is supplied, notification text will be wrapped in an anchor leading to specified address. You can use "Target" key to define a target attribute of the defined link.

Ez a beállítás alapértelmezetten nincs bekapcsolva.

Alapértelmezett érték: 

---
File: <OTRS_CONFIG_Home>/var/notify.txt
Link: https://www.otrs.com
Module: Kernel::Output::HTML::Notification::Generic
Priority: Warning
Target: _blank
Text: The OTRS Website
