On the AdminProcessManagement screen you can find an Example process widget, where you can find best practice example processes.

Following example processes are available in the OTRS Business Solution™:

  • Conference Room Reservation

  • ITIL Incident Management

  • Office Materials Procurement

  • Order Request Management

  • Release And Deployment Management

  • Request For Leave Management

  • Start RMA

  • Travel Expense

3.33. ábra - Import example process widget


To install example process, select desired process from drop-down menu in the Example process widget and click on the Import example process button.

During the import process, system creates needed dynamic fields and/or updates system configuration in order to work properly.
