There are three example web services available in the OTRS Business Solution™:

  • BugzillaConnector

  • JIRAConnector

  • OTRSConnector

To install example web service, go to the GenericInterface Web Service Management page and click on the Add web service button.

3.34. ábra - Add web service


Find Example Web Services widget, select example web service from drop-down menu, and click on the Import example web service button.

3.35. ábra - Import example web service


During the import process, system creates needed dynamic fields and/or updates system configuration in order to work properly.

Note: Example web services requires additional feature add-ons to be installed (OTRSGenericInterfaceInvokerTicket and OTRSGenericInterfaceInvokerEventFilter).
